# Split is real ONLY when the DSO self-binds AND the exe exports its copy.
# Watch the 'register ->' line (which copy the constructor wrote) vs the
# 'get_list <-' line (which copy discovery read).

===== (A) default DSO (no self-bind) + exe EXPORTS static copy =====
# expected: NO split -- constructor is interposed onto the exe copy too
    [constructor in copy=SHARED] registering rxe_train, rxe_store
    [register -> copy=STATIC] this copy's table now holds 1 device(s)
    [register -> copy=STATIC] this copy's table now holds 2 device(s)
    [get_list <- copy=STATIC] this copy holds 2 device(s)
  collective: discovered 2 device(s)

===== (B) DSO -Bsymbolic-functions + exe EXPORTS static copy =====
# expected: SPLIT -- constructor self-binds to the DSO copy; discovery reads the exe copy
    [constructor in copy=SHARED] registering rxe_train, rxe_store
    [register -> copy=SHARED] this copy's table now holds 1 device(s)
    [register -> copy=SHARED] this copy's table now holds 2 device(s)
    [get_list <- copy=STATIC] this copy holds 0 device(s)
  collective: discovered 0 device(s)   *** DEVICE NOT FOUND -- but the constructor DID register devices, into the OTHER copy ***

===== (C) DSO -Bsymbolic (full) + exe EXPORTS static copy =====
# expected: SPLIT too
    [constructor in copy=SHARED] registering rxe_train, rxe_store
    [register -> copy=SHARED] this copy's table now holds 1 device(s)
    [register -> copy=SHARED] this copy's table now holds 2 device(s)
    [get_list <- copy=STATIC] this copy holds 0 device(s)
  collective: discovered 0 device(s)   *** DEVICE NOT FOUND -- but the constructor DID register devices, into the OTHER copy ***

===== (D) DSO -Bsymbolic-functions + exe does NOT export (--exclude-libs,ALL) =====
# expected: NO split -- discovery binds to the DSO copy, same one the constructor wrote
    [constructor in copy=SHARED] registering rxe_train, rxe_store
    [register -> copy=SHARED] this copy's table now holds 1 device(s)
    [register -> copy=SHARED] this copy's table now holds 2 device(s)
    [get_list <- copy=SHARED] this copy holds 2 device(s)
  collective: discovered 2 device(s)

VERDICT: split only in (B) and (C). (A) removes self-binding, (D) removes
the exe's interposition -- either one alone eliminates the split.
